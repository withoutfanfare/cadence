# Cadence Media Pack

Small ready-to-use image set built from the main Cadence app icon. The README
uses the standalone app icon directly; these files are for repository previews,
launch posts, and online placements.

| File | Size | Use |
| --- | --- | --- |
| `cadence-readme-hero.png` | 1400 x 360 | Transparent icon-only README strip |
| `cadence-github-social-preview.png` | 1280 x 640 | GitHub social preview image |
| `cadence-ad-wide-1200x628.png` | 1200 x 628 | Wide social/ad placement |
| `cadence-ad-square-1080.png` | 1080 x 1080 | Square social/ad placement |

The source app icon exports live in `assets/cadence-app-icon/`.
